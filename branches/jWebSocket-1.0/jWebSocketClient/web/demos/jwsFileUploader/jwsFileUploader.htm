This demo shows how to easily upload files with jWebSocket File Uploader PlugIn. This example is based on the File System PlugIn which allows to submit and receive from the server base64 data from a files. This PlugIn also brings a complete HTML 5 File API independency, so, now you will be able to select files, split them and do whatever you want with them by using the jWebSocket Flash File API fallback.
