Here we show how easily jWebSocketReporting can be used to obtain generated reports. You will be able to upload a custom report template, obtain the list of existing templates on server and generate a report by typing your own data.

This demo generates a default report called "person" if you don't type any data in Create Report form. Otherwise if you choose generate your own report you must pick one from the list of reports and introduce the report data as a JSON object as is shown in the next example:
