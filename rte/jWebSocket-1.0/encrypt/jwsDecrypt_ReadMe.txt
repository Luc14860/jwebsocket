Decrpyting a file using a private key
(C) Copyright 2015 Innotrade GmbH (jWebSocket.org), Germany (NRW), Herzogenrath
Authors: Domma Moreno Dager, Alexander Schulze

Required: Your private key <yourname>.key (generated by jwsCreatePrivateKey.bat) in this location
Usage jwsDecrypt.bat on the windows command line: jwsDecrypt.bat <encrypted_target_file> <decrypted_target_file>
Example: c:\>jwsDecrypt.bat myDoc.rar.enc myDoc.rar.dec

You can double click and run jhe jwsDecryptDemo.bat file to decrypt files.
First edit this file and change the name of the parameters by yours, then execute it.
